# Run.nvim
Execute the current file/script

<p>
  <img src="preview.png">

# Test Install
```
  git clone https://github.com/Manas140/run.nvim && cd run.nvim
  nvim --cmd "set rtp+=."
```

# Execute The Plugin
```
  Keybind - Control + E
  Command - :call Run()
```

# Supports
```
  html
  js
  rust
  lua
  python
  c
  cpp
  go
  sh
```
